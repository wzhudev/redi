# Setup Dev Environment

## TypeScript Configuration

Enable decorators in `tsconfig.json`:

```json
{
  "compilerOptions": {
    "experimentalDecorators": true
  }
}
```

<Callout>
  Make sure TypeScript compiles your code, not Babel. If using Babel, enable the `legacy` option in [@babel/plugin-proposal-decorators](https://babeljs.io/docs/babel-plugin-proposal-decorators).
</Callout>

## Starter Template

Get started quickly with [redi-starter](https://github.com/anthropics/redi-starter):

```bash
npx degit anthropics/redi-starter my-project
cd my-project
npm install
```

Features:
- ✅ TypeScript configured for redi
- ✅ Prettier / ESLint / webpack / Jest
- ✅ Platform-specific code loading


## Using Without Decorators

For JavaScript or environments without decorator support, use `setDependencies`:

```js
// Instead of decorators
class MapService {
  constructor(satellite) {
    this.satellite = satellite;
  }
}

setDependencies(MapService, [[new SkipSelf(), ISatelliteService]]);
```
