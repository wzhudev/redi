**redi** (pronounced 'ready') is a lightweight dependency injection library for TypeScript and JavaScript, with React bindings included.

## Quick Start

```bash npm2yarn
npm install @wendellhu/redi
```

```ts
import { Inject, Injector } from '@wendellhu/redi';

class AuthService {
  getCurrentUserInfo(): UserInfo { /* ... */ }
}

class FileListService {
  constructor(@Inject(AuthService) private authService: AuthService) {}
  
  getUserFiles(): Promise<Files> {
    const user = this.authService.getCurrentUserInfo();
    // ...
  }
}

const injector = new Injector([[AuthService], [FileListService]]);
const fileList = injector.get(FileListService);
```

## Features

- **[Dependency Items](/docs/item)**: Class `{ useClass }`, Value `{ useValue }`, Factory `{ useFactory }`, Async `{ useAsync }`
- **[Interface Injection](/docs/identifier)**: Use `createIdentifier` for interface-based injection
- **[Lazy Instantiation](/docs/item)**: Defer creation with `{ lazy: true }`
- **[Hierarchy Injection](/docs/hierarchy)**: Parent-child injectors with `@Self()` and `@SkipSelf()`
- **[Optional & Many](/docs/declare-dependency)**: `@Optional()` and `@Many()` decorators
- **[React Integration](/docs/react)**: `useDependency`, `connectDependencies` and more hooks
- **[RxJS Support](/docs/react)**: `useObservable` and `useUpdateBinder` for reactive programming
